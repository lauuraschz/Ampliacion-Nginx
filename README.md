# Ampliación eventos Nginx

## Descripción

En esta práctica se ha implementado un sistema de detección de eventos en Nginx.

Cada petición HTTP realizada al servidor genera automáticamente una notificación por correo electrónico mediante SMTP utilizando Gmail.

El sistema monitoriza el fichero `/var/log/nginx/access.log` mediante un script Bash.
